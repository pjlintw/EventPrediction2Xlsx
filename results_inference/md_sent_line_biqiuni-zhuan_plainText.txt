1	為法淨尼弟子，住寺，遊學淮海，弘宣正法，先人後己，志在廣濟。
0	以齊建元四年，立法音精舍，禪思靜默，通達三昧，德聲遐布，功化自遠。
0	道俗敬仰，盛修供施。
0	時有慧明法師，深愛寂靜，本住道林寺，永明時，為文惠帝、竟陵文宣王之所修飾。
1	僧多義學，累講經論，去來諠動，明欲去之。
1	簡以寺為施，因移白山，更立草菴，以蔽風雨，應時行乞，取給所資，常聚樵木，云營功德。
0	以建武元年二月十八日夜，登此積薪，引火自焚，捨生死身，供養三寶。
0	近村見火，競來赴救。
0	及至，簡已遷滅。
0	道俗哀慟，聲振山谷，即聚所餘，為立墳剎也。
0	法音寺淨珪尼傳十一
0	淨珪，本姓周，晉陵人也。
0	寓居建康縣三世矣。
0	珪幼而聰頴，一聞多悟，性不狎俗，早願出家。
0	父母怜之，不違其志，為法淨尼弟子，住法音寺。
0	德行純邃，經律博通，三業禪祕，無不善達。
0	神量淵遠，物莫能窺，遺身忘味，常自枯槁。
0	其精進總持，為世法則，傳授訓誘，多能導利，當世歸心。
1	與曇簡尼同憩法音寺，後移白山，栖託樹下，功化轉弘。
0	以建武元年二月八日，與曇簡同夜燒身。
0	道俗哀赴，莫不哽咽，收其舍利，樹封墳剎焉。
0	集善寺慧緒尼傳十二
0	慧緒，本姓周，閭丘高平人也。
0	為人高率疎遠，見之如丈夫，不似婦人。
0	發言吐論，甚自方直，略無所迴避。
0	七歲便蔬食持齋，志節勇猛。
0	十八出家，住荊州三層寺。
0	戒業具足，道俗所美。
0	時江陵有隱尼，西土德望，見緒而異之，遂忘年契意，相携行道。
0	嘗同居一夏，共習般舟，心形勤苦，晝夜不息。
0	沈攸之為刺史，普沙簡僧尼，緒乃避難下都。
0	及沈破敗，後復還西。
1	齊太尉大司馬豫章王蕭嶷，以宋昇明末出鎮荊、陝，知其有道行，迎請入內。
0	備盡四事。
0	時有玄暢禪師，從蜀下荊，緒就受禪法，究極精妙，暢每稱其宿習不淺。
0	緒既善解禪行，兼菜蔬勵節。
0	豫章王妃及內眷屬，敬信甚深，從受禪法。
1	每有嚫施，受已隨散，不嘗儲畜，意志高遠，都不以生業關懷蕭王要共還都，為起精舍，在第東田之東，名曰福田寺，常入第行道。
0	永明九年，自稱忽忽苦病，亦無正惡，唯不復肯食，顏貌憔顇，苦求還寺，還寺即平愈。
0	旬日中，輒復請入，入轉如前，咸不知所以。
0	俄而王薨，禍故相續。
1	武皇帝以東田郊逈，更起集善寺，悉移諸尼還集善，而以福田寺別安外國道人阿梨，第中還復供養，善讀誦呪。
0	緒自移集善寺以後，足不復入第者數年。
0	時內外既敬重此尼，每勸其暫至後第內。
